# VesloBeroun.cz

Website for Veslo Beroun built with Next.js (App Router) and TypeScript.

## Requirements

- Node.js 18+ (recommended: 20+)
- npm (or compatible package manager)

## Getting Started

Install dependencies:

- npm install

Run development server:

- npm run dev

Open http://localhost:3000 in your browser.

## Build

- npm run build
- npm run start

## Project Structure

- src/app — App Router pages and API routes
- src/components — Shared UI components
- src/data — Static data sources
- src/lib — Data access and utilities
- public — Static assets

## Environment Variables

If required, place variables in a .env.local file.

## Scripts

- npm run dev — Start dev server
- npm run build — Build for production
- npm run start — Start production server
- npm run lint — Lint codebase

## Notes

Static assets are served from the public folder. Use absolute paths like /images/hero.jpg.
